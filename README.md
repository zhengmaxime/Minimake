# Minimake

### Author:

* Maxime ZHENG \<maxime.zheng@epita.fr\>

### Build:

```
make
```

### Test:

```
make check
```

### Run:

```
./minimake [options] [target] ...
```
### Additional features:

* Makefile selection
* Self building project
* Comments
* Recursive variables
* Environment variables
* Phony targets
